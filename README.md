# Desenvolvimento de Tema e Plugin Personalizado

## Sobre este projeto

Este repositório contém os arquivos de desenvolvimento personalizados do site do **Museus Castro Maia**, criado como parte do meu trabalho como desenvolvedor no **IBRAM**.

## Tecnologias utilizadas

- WordPress (como CMS base)
- Tema-filho baseado no tema **Bloxy**
- Plugin personalizado para integração de menus institucionais
- Plugins auxiliares:
  - Tainacan
  - Ajax Search
  - PostX
  - Smart Slider 3

## ⚠️ Aviso

Por questões de **confidencialidade e boas práticas**, este repositório contém **apenas os arquivos de minha autoria**, como:

- Tema-filho
- Plugin customizado

Arquivos do core do WordPress, configurações de servidor e banco de dados **não foram incluídos**.

## 🌐 Site em produção

[https://museuscastromaya.museus.gov.br/](https://museuscastromaya.museus.gov.br/)

## 💬 Observações

Este projeto foi desenvolvido em ambiente local utilizando **XAMPP**, com exportação realizada via **All-in-One WP Migration** para ambiente em nuvem (Gov.br/AWS).
